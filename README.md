# FOOD RUSH 🍔⚡

FOOD RUSH is a cinematic full-stack food ordering platform built with React, Node.js, Express, MongoDB, and modern frontend animation systems.

The platform was designed to simulate a real-world restaurant delivery experience with premium UI transitions, interactive cart systems, checkout flows, admin management, and responsive layouts.

---

# 🚀 Features

- Modern animated frontend UI
- Responsive food ordering experience
- Add to cart functionality
- Dynamic cart popup system
- Checkout flow
- Order management
- Admin dashboard
- MongoDB database integration
- REST API backend
- Product management system
- Category filtering
- Interactive hover animations
- Smooth page transitions
- Mobile responsive design
- Full-stack architecture
